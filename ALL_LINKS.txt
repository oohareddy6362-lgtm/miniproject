ğŸ› TROUBLESHOOTING
================================================================================

Database Connection Error:
  Problem: Can't reach database server
  Solution: Update DATABASE_URL in .env file
  Get new DB: https://console.neon.tech/

Port Already in Use:
  Problem: Port 3000 is already in use
  Solution: Run "npx kill-port 3000"

Clerk Authentication Error:
  Problem: Missing or invalid Clerk keys
  Solution: Get new keys from https://dashboard.clerk.com/

Gemini API Error:
  Problem: Invalid API key
  Solution: Get new key from https://makersuite.google.com/app/apikey
